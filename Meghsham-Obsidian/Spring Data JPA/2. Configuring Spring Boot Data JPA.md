Following are steps to work with Spring Bood Data JPA:
1. Create a spring project with Spring Data JPA dependency
2. Configure database connection setting and other customisation  which helps the development in `application.properties`
```properties

# This Enables the colorful logs in console  
spring.output.ansi.enabled=ALWAYS  
  
# Specify database connection url  =, username and password 
spring.datasource.url = jdbc:postgresql://localhost:5432/hospital_jpa  
spring.datasource.username=postgres  
spring.datasource.password=password  

# 
spring.jpa.hibernate.ddl-auto=update  
spring.jpa.show-sql=true  
spring.jpa.properties.hibernate.format_sql =true	
```
