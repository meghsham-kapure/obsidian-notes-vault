### **Configuring Apache Tomcat Server for Running Servlets in Eclipse**

To run Java Servlets in **Eclipse** with **Apache Tomcat**, follow these steps:

---

## **Step 1: Download and Install Apache Tomcat**

1. Go to the official **Apache Tomcat** website:  
    ðŸ”— [https://tomcat.apache.org/](https://tomcat.apache.org/)
2. Download the latest **Tomcat (e.g., Tomcat 9 or 10) ZIP file** for your system.
3. Extract the ZIP file to a directory (e.g., `C:\apache-tomcat-9.0`).

## **Step 2: Configure Tomcat in Eclipse**

1. **Open Eclipse** (Install **Eclipse IDE for Java EE Developers and Web Developers** if not already installed).

## **Step 3: Create a Dynamic Web Project**

1. **Go to Eclipse** â†’ Click **File** â†’ **New** â†’ **Dynamic Web Project**.

## **Step 4: Add a Servlet Class**

1. **Right-click** on the `src` folder â†’ **New** â†’ **Servlet**.
2. Provide a **Servlet Name** (e.g., `HelloServlet`) and **Package Name** (e.g., `com.example`).
3. Click **Next**, then check `doGet()` method, and click **Finish**.
4. Modify the `doGet()` method as shown below:

```java
import java.io.IOException;
import java.io.PrintWriter;
import javax.servlet.ServletException;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;

@WebServlet("/hello")
public class HelloServlet extends HttpServlet {
    protected void doGet(HttpServletRequest request, HttpServletResponse response) 
            throws ServletException, IOException {
        response.setContentType("text/html");
        PrintWriter out = response.getWriter();
        out.println("<h1>Hello, Servlet is Running!</h1>");
    }
}
```

---

## **Step 5: Run the Servlet**

1. **Right-click on the Project** â†’ **Run As** â†’ **Run on Server**.
2. Select **Apache Tomcat** and click **Next**.
3. Choose your project and click **Finish**.
4. Open a browser and enter:
    
    ```
    http://localhost:8080/MyServletApp/hello
    ```
    
    You should see **"Hello, Servlet is Running!"**

---

## **Step 6: Configuring `web.xml` (Optional)**

If you **do not use** the `@WebServlet` annotation, configure the servlet in `web.xml`:

1. Open `WebContent/WEB-INF/web.xml` and add:

```xml
<servlet>
    <servlet-name>HelloServlet</servlet-name>
    <servlet-class>com.example.HelloServlet</servlet-class>
</servlet>
<servlet-mapping>
    <servlet-name>HelloServlet</servlet-name>
    <url-pattern>/hello</url-pattern>
</servlet-mapping>
```

2. Restart Tomcat and access the servlet at:
    
    ```
    http://localhost:8080/MyServletApp/hello
    ```

### **Troubleshooting**

- **Port 8080 Already in Use?** Change the port in `server.xml`:
    - Go to `Tomcat Installation Directory/conf/server.xml`.
    - Find `<Connector port="8080" protocol="HTTP/1.1"/>`.
    - Change `8080` to `8081` or any available port.
- **404 Error?** Ensure the servlet class or URL mapping is correct.
- **Class Not Found?** Ensure Tomcat is properly added to the project's runtime.
