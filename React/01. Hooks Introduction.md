React Hooks are tools using we can use React Features like state and effect in inside the function component [which was only available for class components]

**React Hooks** are functions that let you "hook into" React state and life-cycle features from function components.
### Why were Hooks made?

1. **Make code simpler** – No need for classes.
2. **Reuse logic easily** – Share state/effects between components.
3. **Fix messy lifecycle** – Old way was confusing.
4. **Write less, do more** – Cleaner and shorter code
